# Healthcare Claims Medallion Lakehouse Pipeline

Enterprise-style Azure Databricks project for healthcare claims ingestion, validation, and analytics using Medallion Architecture.

- Databricks
- PySpark
- Delta Lake
- Unity Catalog
- GitHub

- Bronze
- Silver
- Gold
- Reject

- Audit and traceability
- Incremental processing

- Data quality checks
- Reject handling
